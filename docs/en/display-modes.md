# Tokénmon Display Modes

[← Generations](generations.md) · [Back to README](../../README.md)

## Recommended mode

Braille is the primary supported display mode and should be the default recommendation for most users. It works across the widest set of terminals and matches the most stable path in the current project.

## Other modes

Tokénmon still exposes additional renderer paths and display combinations, but they should be treated as more experimental than Braille. If you hit rendering issues, switch back to Braille first.

## Useful config values

- `renderer braille`
- `sprite_mode all`
- `sprite_mode ace_only`
- `sprite_mode emoji_all`
- `sprite_mode emoji_ace`
- `info_mode ace_full`
- `info_mode name_level`
- `info_mode all_full`
- `info_mode ace_level`
