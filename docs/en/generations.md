## Multi-generation support

Tokénmon is no longer framed as a single-generation project. The repository carries Pokémon data from multiple generations, and the active generation changes the progression context you see in the CLI, status line, encounters, and Pokédex views.

## What changes with the active generation

- starter pools and available species data
- regional progression and encounter tables
- generation-specific Pokédex progress
- the collection context reflected in CLI views

## Switching guidance

Use the generation commands in the CLI to inspect or change your active generation. After switching, review your current party, region, and Pokédex views so the new context is clear before continuing a long session.
