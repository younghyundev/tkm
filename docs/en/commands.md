# Tokénmon Commands

[← Gameplay Systems](systems.md) · [Next: Generations →](generations.md)

## Setup and status

- `/tkm:setup`
- `/tkm:tkm status`
- `/tkm:tkm dashboard`
- `/tkm:tkm stats`

## Party and collection

- `/tkm:tkm starter`
- `/tkm:tkm party`
- `/tkm:tkm box`
- `/tkm:tkm pokedex`
- `/tkm:tkm achievements`
- `/tkm:tkm items`
- `/tkm:tkm notifications`

## Party management

- `/tkm:tkm party add <name>`
- `/tkm:tkm party remove <name>`
- `/tkm:tkm party dispatch <name>`
- `/tkm:tkm party swap <slot> <name>`
- `/tkm:tkm party reorder <from> <to>`
- `/tkm:tkm party suggest`

## Progression and world

- `/tkm:tkm evolve`
- `/tkm:tkm region`
- `/tkm:tkm region list`
- `/tkm:tkm region move <name>`
- `/tkm:tkm gen`
- `/tkm:tkm guide`
- `/tkm:tkm legendary`

## Config

- `/tkm:tkm config set language <en|ko>`
- `/tkm:tkm config set renderer braille`
- `/tkm:tkm config set sprite_mode all`
- `/tkm:tkm config set sprite_mode ace_only`
- `/tkm:tkm config set info_mode ace_full`
- `/tkm:tkm config set info_mode name_level`
