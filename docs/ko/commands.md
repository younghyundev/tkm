# Tokénmon 명령어

## 설정과 상태

- `/tkm:setup`
- `/tkm:tkm status`
- `/tkm:tkm dashboard`
- `/tkm:tkm stats`

## 파티와 수집

- `/tkm:tkm starter`
- `/tkm:tkm party`
- `/tkm:tkm box`
- `/tkm:tkm pokedex`
- `/tkm:tkm achievements`
- `/tkm:tkm items`
- `/tkm:tkm notifications`

## 파티 관리

- `/tkm:tkm party add <이름>`
- `/tkm:tkm party remove <이름>`
- `/tkm:tkm party dispatch <이름>`
- `/tkm:tkm party swap <슬롯> <이름>`
- `/tkm:tkm party reorder <from> <to>`
- `/tkm:tkm party suggest`

## 진행과 월드

- `/tkm:tkm evolve`
- `/tkm:tkm region`
- `/tkm:tkm region list`
- `/tkm:tkm region move <이름>`
- `/tkm:tkm gen`
- `/tkm:tkm guide`
- `/tkm:tkm legendary`

## 설정

- `/tkm:tkm config set language <en|ko>`
- `/tkm:tkm config set renderer braille`
- `/tkm:tkm config set sprite_mode all`
- `/tkm:tkm config set sprite_mode ace_only`
- `/tkm:tkm config set info_mode ace_full`
- `/tkm:tkm config set info_mode name_level`
