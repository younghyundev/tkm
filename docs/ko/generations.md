# Tokénmon 세대

[← README로 돌아가기](../../README.ko.md)

## 다세대 지원

Tokénmon은 더 이상 단일 세대 프로젝트로만 소개되지 않습니다. 저장소에는 여러 세대의 포켓몬 데이터가 들어 있으며, 활성 세대는 CLI, 상태 줄, 인카운터, 도감 화면에서 보이는 진행 문맥을 바꿉니다.

## 활성 세대에 따라 바뀌는 것

- 스타터 풀과 사용 가능한 종 데이터
- 지역 진행과 인카운터 테이블
- 세대별 도감 진행도
- CLI 화면에 반영되는 수집 문맥

## 전환 가이드

CLI의 세대 관련 명령으로 현재 활성 세대를 확인하거나 변경할 수 있습니다. 세대를 바꾼 뒤에는 파티, 지역, 도감 화면을 다시 확인해 새로운 문맥을 먼저 파악하는 것이 좋습니다.
