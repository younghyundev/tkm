# Tokénmon 표시 모드

[← README로 돌아가기](../../README.ko.md)

## 권장 모드

Braille은 현재 Tokénmon에서 가장 우선적으로 권장되는 표시 모드입니다. 가장 넓은 범위의 터미널에서 동작하며, 현재 프로젝트에서 가장 안정적인 경로와도 맞닿아 있습니다.

## 다른 모드

Tokénmon에는 추가 렌더링 경로와 표시 조합도 남아 있지만, Braille보다 실험적인 선택지로 보는 편이 맞습니다. 렌더링 이슈가 생기면 먼저 Braille로 되돌리는 것을 권장합니다.

## 자주 쓰는 설정값

- `renderer braille`
- `sprite_mode all`
- `sprite_mode ace_only`
- `sprite_mode emoji_all`
- `sprite_mode emoji_ace`
- `info_mode ace_full`
- `info_mode name_level`
- `info_mode all_full`
- `info_mode ace_level`
