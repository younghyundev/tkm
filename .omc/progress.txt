### Key Learnings
1. postinstall must be self-contained — cannot import from other src/ modules during npm install
2. Node 22 tsconfig: module must be Node16 when moduleResolution is Node16
3. Node 22 TypeScript stripping: `!!` in -e flags gets mangled by shell+node type stripper
4. pngjs has no built-in types — need @types/pngjs
5. Config `as Record<string, unknown>` fails strict mode — use `as any` for dynamic key access
